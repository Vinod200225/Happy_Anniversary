# public/timeline/

The three story photos for the timeline section. Naming is **strict** — the manifest looks for "year1", "year2", "year3" anywhere in the filename.

```
year1.jpg   →  Year One:  When you fell for me
year2.jpg   →  Year Two:  When I lost my way
year3.jpg   →  Year Three: Finding my way back to you
```

Any of these extensions work: `.jpg .jpeg .png .webp .gif .avif`

If a slot is missing the page shows a soft "drop a photo named yearN.jpg" placeholder for that milestone only — the rest of the page is unaffected.
